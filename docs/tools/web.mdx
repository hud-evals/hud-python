Web access comes in two forms: hosted tools the provider executes server-side,
and browser automation your environment exposes as a `cdp` capability.

## Quick Reference

| Tool | Execution | Purpose |
|------|-----------|---------|
| `ClaudeWebSearchTool` | Hosted (Claude) | Real-time web search |
| `GeminiGoogleSearchTool` | Hosted (Gemini) | Google search |
| `ClaudeWebFetchTool` | Hosted (Claude) | Fetch page content |
| `cdp` capability | Environment | Full browser automation |

## ClaudeWebSearchTool

Claude's native web search. Executed server-side by Anthropic. Results appear in
the response with citations.

```python
from hud.agents.claude import ClaudeAgent, ClaudeWebSearchTool

agent = ClaudeAgent.create(
    hosted_tools=[
        ClaudeWebSearchTool(
            max_uses=10,
            allowed_domains=["docs.python.org"],
            blocked_domains=["spam.com"],
        )
    ]
)
```

Uses `web_search_20250305` API. $10 per 1,000 searches.

<Note>
Hosted tools are configured on the agent because the provider executes them.
They are not MCP environment tools and are not called through `ctx.call_tool`.
</Note>

## GeminiGoogleSearchTool

Google Search for Gemini. Also hosted and executed by Google.

```python
from hud.agents.gemini import GeminiAgent, GeminiGoogleSearchTool

agent = GeminiAgent.create(hosted_tools=[GeminiGoogleSearchTool()])
```

## ClaudeWebFetchTool

Claude hosted web fetch for URLs and PDFs.

```python
from hud.agents.claude import ClaudeAgent, ClaudeWebFetchTool

agent = ClaudeAgent.create(
    hosted_tools=[ClaudeWebFetchTool(max_content_tokens=20_000)]
)
```

## Browser Automation

For full browser control — navigation, clicks, form filling, screenshots — run
Chromium with remote debugging in your environment and declare a `cdp`
capability. The agent harness drives the browser; you don't register a tool.

```python
from hud import Environment
from hud.capabilities import Capability

env = Environment("web-env")
# Chrome running with: --remote-debugging-port=9222
env.add_capability(Capability.cdp(url="http://localhost:9222"))
```

<Note>
v5's `PlaywrightTool` is removed. Existing v5 environments that registered it
resolve to a no-op; declare a `cdp` capability instead.
</Note>
