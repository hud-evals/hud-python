The HUD platform at [hud.ai](https://hud.ai) is your central dashboard for managing AI agent infrastructure. Browse and train models, deploy environments, organize tasks into evaluation sets, and analyze results.

## Getting Started

Sign in at [hud.ai](https://hud.ai) with your account. The main navigation gives you access to:

- **Home** ‚Äî View recent jobs and traces
- **Models** ‚Äî Browse models, view checkpoints, train custom models
- **Environments** ‚Äî Deploy and monitor your agent environments
- **Tasksets** ‚Äî Organize tasks for evaluations and benchmarks
- **üîç MCP** ‚Äî Connect your AI agent to query platform data

## Home Dashboard

Your home page at [hud.ai/home](https://hud.ai/home) shows:

- **Recent Jobs** ‚Äî Evaluation jobs currently running or recently completed
- **Latest Traces** ‚Äî Your latest task runs with outcomes, filterable by user and searchable

## API Keys

Get your API key at [hud.ai/settings/api-keys](https://hud.ai/settings/api-keys). You'll need this for:

- Gateway requests via `inference.hud.ai`
- CLI authentication (`hud set HUD_API_KEY=...`)
- SDK initialization

## Team Settings

Manage your team at [hud.ai/settings](https://hud.ai/settings):

- **API Keys** ‚Äî Create and rotate keys
- **Team Members** ‚Äî Invite collaborators
- **Billing** ‚Äî View usage and manage subscription
