HUD is a platform for building RL environments for AI agents. Turn your code into agent-callable tools, define evaluation logic, and run evals or training at scale with the same API.

## Install

```bash
# Install CLI
uv tool install hud-python --python 3.12

# Set your API key
hud set HUD_API_KEY=your-key-here
```

Get your API key at [hud.ai/project/api-keys](https://hud.ai/project/api-keys).

## Core Concepts

HUD has four building blocks. Everything else is built on top of them.

**Environments** — the world an agent lives in. Packages tools and scenarios into a deployable unit.

**Tools** — functions agents can call. Decorate with `@env.tool()` and agents see the signature, docstring, and return type.

**Scenarios** — define how agents are evaluated. An async generator with two yields: first sends a prompt, second scores the result.

**Tasks** — a scenario with specific arguments. Group tasks into tasksets for benchmarking and training.

Here's the simplest environment in HUD:

```python
from hud import Environment

env = Environment("my-env")

@env.tool()
def search(query: str) -> str:
    """Search the knowledge base."""
    return db.search(query)

@env.scenario("find-answer")
async def find_answer(question: str):
    answer = yield f"Find the answer to: {question}"
    yield 1.0 if "correct" in answer.lower() else 0.0

task = env("find-answer", question="What is the capital of France?")
result = await task.run("claude-sonnet-4-5")
print(f"Reward: {result.reward}")
```

## The Workflow

```bash
hud init my-env              # Scaffold environment
# Edit env.py                # Add tools and scenarios
hud dev env:env -w env.py    # Run MCP server with hot-reload
hud eval tasks.py            # Run evals locally
hud deploy                   # Deploy to platform, run evals and training at scale
```

## Run Agents

Use any model through one OpenAI-compatible endpoint at `inference.hud.ai`. Create tasksets from your scenarios, run evaluations across models, and train using RL on your tasksets through integrations with Tinker, OpenAI, and Coreweave.

Browse models at [hud.ai/models](https://hud.ai/models).
